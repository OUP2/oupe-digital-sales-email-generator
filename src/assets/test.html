INFORMACIÓN PARA PROFESORES

Debes enviarlos a padres y/o alumnos; puedes descargar una plantilla aquí.

  1. Para acceder a la versión digital del libro del alumno, accede a Oxford Premium,
    copia el código de licencia que aparece al pulsar el botón “Código OLB profesor”

  2. Accede a Oxford Learner’s Bookshelf y usa el “Código OLB profesor”

    de recursos para profesor incluidos en el adjunto de este email.

    para activar el código de licencia que les has enviado previamente.
